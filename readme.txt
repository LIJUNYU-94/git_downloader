===============================
 GitHub 自動ダウンロードツール
===============================

このツールは、GitHub のプロジェクトを **自動でダウンロード＆展開** することができる。

**使い方**
-----------------------
1️⃣ **`download.exe` を開く**

2️⃣ **GitHub のリンクを貼り付け、Enterで続く**
   - 例: https://github.com/user/repository

3️⃣ **フォルダ名を入力し、Enter を押す**
   - **命名ルール**: フォルダ名は **学籍番号の下2桁** を入力してください
